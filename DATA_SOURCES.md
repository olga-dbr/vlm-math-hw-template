# Источники данных для математического направления

Этот репозиторий содержит только маленький toy-набор. Для настоящей версии курса лучше использовать профильные источники по визуально-математическому рассуждению.

## Рекомендуемая схема

| Назначение | Источник | Что брать |
|---|---|---|
| Public toy/smoke | Синтетические задачи преподавателя | 50–500 простых примеров |
| Adapter alignment | MAVIS-Caption или synthetic diagram captions | диаграммы + описания |
| SFT | MAVIS-Instruct / MathVista-style subset | изображение + вопрос + ответ |
| Public dev | MathVista testmini-like curated subset | без hidden labels |
| Hidden eval | MATH-Vision / We-Math-style curated subset | геометрия, графики, диаграммы |

## Замечания по лицензиям

Перед публикацией датасета в курсе проверьте:

- можно ли использовать исходный набор в образовательных целях;
- можно ли делать производный subset;
- можно ли выкладывать изображения студентам;
- нужно ли закрывать репозиторий с данными;
- можно ли публиковать leaderboard.

## Почему не Marketing/MMMU как основной benchmark

MMMU полезен как широкий college-level multimodal benchmark, но для направления математиков основная проверка должна быть ближе к математическому рассуждению: geometry, plots, algebra, combinatorics, functions, diagrams.
